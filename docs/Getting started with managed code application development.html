Getting started with managed code application development

Applies to Dynamics 365 for Customer Engagement apps version 9.x

This guide shows you how to write application code that connects to the Dynamics 365 for Customer Engagement web services and invoke web methods to perform common business data operations like create, delete, update, and find. You will also be shown how to set up a project in Visual Studio with the required namespaces, and how to handle exceptions returned from the web services. After completing this guide, you will have learned enough information to explore the many other business operations that are supported by the web services.
